[官方脚手架](https://uniapp.dcloud.io/worktile/CLI.html#uni-cli)
```
npm install -g @vue/cli // 全局安装脚手架
vue create -p dcloudio/uni-preset-vue my-project // 创建下面
cd my-project
yarn serve
```
