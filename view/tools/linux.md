 ``sudo`` linux系统管理指令，让普通用户有系统管理员执行一些或者全部的root命令的一个工具

 ``systemctl`` Systemd 的主命令,用于管理系统。

（rm: remove、dir: directory、mk: make、cd: Change Directory切换目录）


#### 目录
 ``mkdir`` 创建目录（mkdir gg）

 ``rmdir`` 删除目录

 ``ls`` 查看当前目录文件

 ``cd`` 进入子目录(Change Directory切换目录)




#### 文件
 ``touch``  创建文件

 ``rm`` 删除文件

 ``vi``    编辑文件（i写入）

 ``:wq!``   保存退出(先按"ESC"键)

 ``:q!``    退出





#### 启动服务
``systemctl start NetworkManager`` 开启网络
